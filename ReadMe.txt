This is a collection of code was desgined to solve different problems related to the KNN classifier, this code was used by the following papers:

1- Hassanat, A. B., Abbadi, M. A., Altarawneh, G. A., & Alhasanat, A. A. (2014). Solving the problem of the K parameter in the KNN classifier using an ensemble learning approach. arXiv preprint arXiv:1409.0919.
2- Ahmad Hassanat: Dimensionality Invariant Similarity Measure. Journal of American Science. 08/2014; 10(8):221-226.
3- Abu Alfeilat, H. A., Hassanat, A. B., Lasassmeh, O., Tarawneh, A. S., Alhasanat, M. B., Eyal Salman, H. S., & Prasath, V. S.  Effects of Distance Measure Choice on K-Nearest Neighbor Classifier Performance: A Review. Big Data.7 (3), 2019,
4-	Ahmad B. A. Hassanat, Greedy Algorithms for Approximating the Diameter of Machine Learning Datasets in Multidimensional Euclidean Space: Experimental Results, Advances in Distributed Computing and Artificial Intelligence Journal, 7(3), 2018
5-	Ahmad B. A. Hassanat, Norm-Based Binary Search Trees for Speeding Up KNN Big Data Classification, Computers, MDPI, 7(4), 2018.
6-	Ahmad B. A. Hassanat, Two-Point-Based Binary Search Trees for Accelerating Big Data Classification using KNN, PLOS ONE, 2018.
7-	Ahmad B. A. Hassanat, Furthest-pair-based binary search tree for speeding big data classification using k-nearest neighbors, Big Data, 6(3), 2018.
8-	Ahmad B. Hassanat, Furthest-Pair-Based Decision Trees: Experimental Results on Big Data Classification, Information, MDPI, 9(11), 2018.
